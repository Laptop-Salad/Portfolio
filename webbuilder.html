About Web Builder

Live Site GitHub

Web builder is a site where you can drag and drop elements and generate the code to create your very own site in minutes!
This project is in-progress and is being built using HTML, CSS, JS and jQuery.
As of current below are the main features:

  • All headings are available to use
  • "Infinite" canvas
  • Drag and drop elements within canvas
  • Generate full code
  • Change properties of elements
