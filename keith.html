About Keith

Live Site GitHub

Keith is a CSS library with duck-related colours. This library is highly customisable and it is made easy for users to use their own colour palette or to change the mechanics.

Keith is an in-progress project and is built using CSS, JS and PrismJS. PrismJS is a library to allow you to display beautiful code snippets.

As of current below are the components available:

  • Small, medium and large buttons
  • Progress bars
  • Info, yes/no modal
  • Accordions
