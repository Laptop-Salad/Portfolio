About Spectrum Debate

Live Site GitHub

Spectrum Debate is a debate platform where users can vote based on whether they agree, disagree of feel neutral about a statement.

This project was built using PHP, SQL, JS, jQuery, HTML, CSS, Bootstrap and a mix of Cypress and unit testing was used. Cypress is used for front-end testing.

Here are the main features:

  • Sign up / Log in
  • Creating/Editing/Deleting statements
  • Creating/Editing/Deleting comments
  • Account deletion
  • Voting on statements
  • Viewing user profiles with all their statements and comments
  • Searching for statements
  • Searching for user profiles
  • Adding users as friends
  • Attaching images to statements
  • Users can add a bio to their profile
